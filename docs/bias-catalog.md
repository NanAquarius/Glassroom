# Bias catalog

Glassroom's current built-in cognitive trap catalog is designed for structured intelligence analysis rather than generic productivity advice.

## Source basis

This catalog is informed by public intelligence-analysis tradecraft material, especially:

- CIA — *Psychology of Intelligence Analysis*  
  <https://www.cia.gov/resources/csi/books-monographs/psychology-of-intelligence-analysis-2/>
- CIA — *Tradecraft Primer: Structured Analytic Techniques for Improving Intelligence Analysis*  
  <https://www.cia.gov/resources/csi/static/Tradecraft-Primer-apr09.pdf>
- ODNI — analytic objectivity / ICD 203 analytic standards overview  
  <https://www.dni.gov/index.php/how-we-work/objectivity>

## Built-in bias types

### confirmation bias

- Risk: preferred narratives filter incoming evidence.
- Default mitigation: contradiction-first reading plus analysis of alternatives.

### anchoring

- Risk: early frames or first numbers dominate later judgment.
- Default mitigation: remove the original anchor and compare alternative framings.

### mirror imaging

- Risk: the analyst projects their own motives or logic onto another actor.
- Default mitigation: rebuild the judgment from the actor's incentives and constraints.

### availability heuristic

- Risk: vivid or recent evidence gets overweighted.
- Default mitigation: reweight evidence by diagnostic value, not salience.

### overconfidence

- Risk: certainty outruns the evidence base.
- Default mitigation: separate facts, assumptions, and judgments, then recalibrate confidence.

### groupthink

- Risk: consensus forms before alternatives are genuinely tested.
- Default mitigation: devil's advocacy or red-team review.

### wishful thinking

- Risk: desirable outcomes are mistaken for likely outcomes.
- Default mitigation: premortem analysis.

### premature closure

- Risk: analysis stops before key uncertainties are checked.
- Default mitigation: list unresolved questions and define a stop threshold.

### framing effect

- Risk: wording or presentation biases the judgment path.
- Default mitigation: reframe the problem in multiple ways and compare the conclusions.

### satisficing

- Risk: the first acceptable explanation is treated as sufficient.
- Default mitigation: force one more serious competing hypothesis.

### recency bias

- Risk: the newest signal dominates the baseline.
- Default mitigation: restore longer-term trend and baseline comparison.

### fundamental attribution error

- Risk: intent is over-attributed while situational constraints are underweighted.
- Default mitigation: compare motive-based explanations with structural and organizational ones.
