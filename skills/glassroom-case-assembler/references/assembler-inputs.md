# Assembler inputs

## Base case

Optional but recommended.

Use it to provide:

- title
- context
- geopoliticalContext
- keyActors
- policyQuestion
- assumptions
- discussionPrompts
- expertAnalysis

## Source card

Adds:

- sourceCards
- possibly a better title if base case is missing one

## Bias analysis

Adds:

- biases
- notes that can inform expertAnalysis

## Mitigation pack

Adds:

- mitigations
- provenance / pitfall-related action structure

## Structured analysis

Adds:

- structuredAnalysis
- possible refinement of policyQuestion
