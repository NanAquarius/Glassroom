# Source search playbook

Use this playbook when the user asks for original materials, primary texts, or professional-source discovery.

## 1. Start from the case question

Before searching, reduce the case to:

- actor
- decision or judgment
- timeframe
- source family likely to contain originals

Example:

- actor: U.S. diplomats
- decision area: Cuba normalization expectations
- timeframe: late 2000s
- likely source family: diplomatic cables

## 2. Search upstream first

Try the most likely source family before broad web search.

### Diplomatic / historical cases
Prefer:
- FRUS
- National Archives
- WikiLeaks PlusD

### Intelligence / assessment cases
Prefer:
- CIA Reading Room
- National Security Archive

### Policy / legal / public decision cases
Prefer:
- official government sites
- international organizations
- court / legal repositories where relevant

## 3. Only then widen outward

Use broader search when:

- the archive search returns nothing
- you need context for naming, dates, actors, or terminology
- you need to discover the exact official document title

## 4. Output ranked candidates

When discovery is part of the task, return a small ranked list:

- Tier 1: strongest primary material
- Tier 2: official context material
- Tier 3: secondary framing material

## 5. Common failure modes

### Failure mode: using a summary as if it were original evidence
Fix: explicitly label it as secondary and keep searching upstream.

### Failure mode: stopping at the first searchable source
Fix: ask whether the case would be stronger with a cable, archive record, or official statement.

### Failure mode: mixing provenance and interpretation
Fix: separate:
- what the original record says
- what later analysts say it means

## 6. Recommended search phrasing patterns

### For diplomatic cables
- exact actor + event + cable
- subject phrase in quotes
- actor + timeframe + embassy / state department / cable

### For official statements
- institution + issue + press release
- institution + issue + statement
- site:gov issue title

### For academic framing
- case name + intelligence analysis
- case name + cognitive bias
- case name + foreign policy decision making
