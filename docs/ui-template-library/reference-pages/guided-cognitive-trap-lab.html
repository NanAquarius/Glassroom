1Context Understanding / 背景理解
Learning objective
Establish the geopolitical context, the actors, and the decision frame before discussing cognitive traps.
  • What historical or institutional context shaped the judgment?
  • What pressures were operating on the analysts or decision-makers?
  • What exactly was being asked or decided?
2Identify Assumptions / 识别假设
Analysis task
Make the assumptions visible before judging whether the interpretation was sound.
  • Assumption A about actor intent
  • Assumption B about institutional constraints
  • Assumption C about likely response patterns
3Evaluate Evidence / 证据评估
Analysis task
Separate supporting evidence, contradicting evidence, and ignored warning signs so interpretation never outruns the record.
✔ Supporting evidence / 支持证据
  • Evidence point that appears to support the initial judgment
  • Another supporting signal, ideally specific and attributable
✘ Contradicting evidence / 反驳证据
  • Contradictory evidence that weakens the preferred reading
  • An indicator that should have triggered more caution
⚠ Ignored warning signs / 被忽视的信号
  • Important but discounted warning sign
  • Missing evidence that would have changed confidence
4Cognitive Trap Detection / 认知陷阱识别
Analysis task
Use bias badges for fast recognition, then connect each one to a short explanation card.

5Mitigation Design / 缓解设计
Analysis task
Mitigations should be concrete analytical moves, not generic advice.
Key assumptions check
Force explicit testing of the assumptions that made the original judgment seem persuasive.
Alternative analysis / red team
Make one serious competing interpretation visible before final synthesis.
★Expert Analysis / 专家解析
Reveal after attempt
This section should read like a walkthrough that rewards independent analysis, not like the first thing the learner sees.
Expert walkthrough

Use this area for a concise synthesis that connects context, assumptions, evidence imbalance, identified traps, and better analytic moves.
