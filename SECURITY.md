# Security Policy

## Supported versions

Glassroom is currently maintained on the latest published release line.

## Reporting a vulnerability

Please do not open a public issue for sensitive security reports.

Instead, contact the maintainer privately first and include:

- affected component
- impact summary
- reproduction steps
- any known mitigation

## Scope notes

Glassroom intentionally focuses on public-safe workflow code and de-identified artifacts.

Please do not submit:

- private classroom material
- secrets or credentials
- personally identifying data
- copyrighted assets you do not have permission to disclose
