# Technique selector

## Use ACH when

- there are multiple competing hypotheses
- the task is to compare explanations, not just summarize

## Use devil's advocacy when

- one interpretation dominates too early
- the point is to challenge a comfortable narrative

## Use key assumptions check when

- the analysis depends on a few fragile assumptions

## Use indicators / signposts when

- the task is forward-looking
- the user wants to monitor future developments

## Use chronology when

- confusion comes from event order or causal sequencing
