# Task routing

## Use `glassroom-source-intake`

When the user asks:

- what is this source
- where is this material from
- classify these files
- extract metadata from case documents

## Use `glassroom-cognitive-bias`

When the user asks:

- what biases are in this text
- analyze cognitive traps
- explain mirror imaging / anchoring / confirmation bias in this case

## Use `glassroom-structured-analysis`

When the user asks:

- apply ACH
- challenge assumptions
- compare alternative hypotheses
- build indicators / signposts

## Use `glassroom-case-html`

When the user asks:

- turn this case into an HTML page
- make an interactive classroom/demo page
- build a bias explorer from this material

## Use `glassroom-course-writing`

When the user asks:

- write the memo
- turn analysis into an essay
- produce a discussion post or presentation script
